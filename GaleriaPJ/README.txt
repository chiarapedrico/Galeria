Galeria Processing

Feito por: Chiara Pedrico

Para mover a galeria pelo teclado aperte(a cada movimento de um clique):
A - direita
D - esquerda
W - Para ir para frente
S - para ir para trás
Q - para descer
E - para subir

Para dar zoom e tirar o zoom role o scroll do mouse.

Para mudar os ângulos clique no mouse e arraste.
